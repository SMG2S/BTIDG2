Quick Start¶

Dependencies¶

The dependencies are:

  • a C compiler

  • CMake >= 3.18.4

  • MPI (Message Passing Interface)

Build¶

  1. Get the code from github

git clone https://github.com/SMG2S/BTIDG2.git

  1. get into the project folder

cd btidg2

  1. make a directory for building and compiling, then change into this new built folder

mkdir build
cd build

  1. build the code with CMake

cmake ..
make -j

  1. in the folder example, two executable binaries are available, and the execution is:

mpirun -n ${NPROCS} ./example/BTIG2_A -n ${N}

and

mpirun -n ${NPROCS} ./example/BTIG2_AT -n ${N}

in which ${NPROCS} is the number of MPI processes to be used, and ${N} is the size of squared to be generated.
